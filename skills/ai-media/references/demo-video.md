# Demo Video

Generate product demo videos programmatically with Remotion + voiceover.

## What This Does
- Analyze app structure + feature description
- Generate tight narration script
- Capture screenshots or short recordings
- Generate voiceover via ElevenLabs when needed
- Compose scenes with Remotion
- Render to MP4

## Usage
```
/ai-media demo [product] [feature to demo]
/ai-media demo caesar "onboarding flow" with voiceover
```

## Output
`demo-[product]-[feature].mp4` in current directory

## Prerequisites
- Remotion skills loaded
- Optional: ElevenLabs for voiceover
